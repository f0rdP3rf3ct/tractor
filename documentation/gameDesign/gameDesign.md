# Tractor2 (working title) / non-stop-crops

## Target Audience
Children ages 4 and up

## Basic Concept
An isometric endless scroller in which players use interchangeable vehicles to harvest a given field. The time taken is displayed at the end. The focus is on having fun. Competitive gameplay is not necessary; at most, players can compare their performance against their own previous scores.
The field size should be adjustable as a setting before playing. The default setting is a 10x10 tile field.
Once all tiles have been harvested, a rewarding completion animation plays and the game can be started again.
The vehicle can be changed at any time during gameplay.

## Distribution / Publication
The game is a simple web application that can also be used as a PWA and should be playable on mobile devices (smartphone, iPad) as well as as a desktop application using keyboard or Bluetooth-capable controllers.
On mobile, touch controls must be supported — most simply with four arrow buttons, so that it is easy for children to understand.
The game should ultimately be available for free on various platforms and published on itch.io.

# Further Ideas
- To add more variety to the game, there should be cosmetic objects such as scarecrows or animals on the field.
- There should also be obstacles (trees, buildings).
